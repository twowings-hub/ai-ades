AI-ADES 사용자 매뉴얼

CO₂ 레이저 Film 절단 최적화 시스템 — 운영자를 위한 화면별 상세 안내서

이 매뉴얼의 목적: AI-ADES를 처음 사용하는 운영자도 각 화면에서 무엇을 입력하고, 어떤 값을 확인하며, 어떤 버튼을 눌러야 하는지 명확히 이해할 수 있도록 안내합니다.

AI-ADES란 무엇인가

AI-ADES는 CO₂ 레이저로 Glass + Film 적층 소재를 절단할 때, 최적의 레이저 조건을 AI가 자동으로 찾아주는 시스템입니다.

🎯 핵심 가치
항목기존 방식AI-ADES

운영자가 해야 할 일

AI-ADES에서 운영자는 딱 세 가지만 하면 됩니다.

  1. 소재 정보 입력 (M1 길이, M2 길이, Thickness)
  2. AI 제안 검토 후 승인 또는 거부
  3. 가공 후 계측값 입력 (Kerf, Depth)

레이저 파라미터 계산은 AI가 담당합니다.

    AI가 제안한 Speed, Defocus, Frequency, Power 값과 예측 결과(Depth, 판정, 신뢰도)를 확인합니다. SHAP 차트와 AI 설명도 함께 표시됩니다.

    계측값 입력

입력 항목 상세

  • M1 (Glass) 길이 mm
    첫 번째 소재인 Glass의 센싱 길이입니다. 레이저가 먼저 통과하는 물질입니다. 4mm / 10mm / 20mm 중 하나를 버튼으로 선택합니다. 현재 가공할 Glass 시편의 규격을 확인하여 선택하세요.
  • M2 (Film) 길이 mm
    두 번째 소재인 Film의 센싱 길이입니다. PET(75μm) + 접착층(25μm)이 적층된 소재입니다. 10mm / 25mm / 50mm 중 하나를 버튼으로 선택합니다. AI가 이 값을 기준으로 최적 Power 범위를 계산합니다.
  • Thickness (실측값) μm
    Film(M2)의 실제 두께를 현미경으로 측정한 값입니다. 목표값은 100μm이지만 실제로는 ±20μm 변동이 있습니다. 반드시 가공 전에 현미경으로 직접 측정한 값을 입력하세요. 이 값이 부정확하면 AI 예측 정확도가 낮아집니다. 소수점 한 자리까지 입력 가능합니다(예: 105.4).
중요: Thickness는 매 가공 세션마다 새로 측정하여 입력해야 합니다. 전날 값을 그대로 사용하면 AI 예측 오차가 커질 수 있습니다.

AI 추천 파라미터 — 각 항목 설명

  • Speed mm/s
    레이저 스테이지의 이동 속도입니다. 200 / 500 / 1000 mm/s 세 값 중 하나로 제안됩니다. 속도가 빠를수록 단위 면적당 에너지가 줄어들어 덜 깊이 가공됩니다.
  • Defocus mm
    레이저 초점이 소재 표면에서 Z축 방향으로 벗어난 거리입니다. 0 / 1 / 2 / 3 / 4 mm 중 하나로 제안됩니다. 0이면 초점이 정확히 맞은 상태이고, 값이 클수록 초점이 흐려져 에너지 밀도가 낮아집니다.
  • Frequency kHz
    레이저 펄스 주파수입니다. 100 kHz 또는 200 kHz 중 하나로 제안됩니다. 주파수가 높을수록 단위 시간당 펄스 수가 많아집니다.
  • Power W
    레이저 출력값입니다. 2.8W ~ 59.8W 사이에서 연속적으로 제안됩니다. AI가 가장 세밀하게 조정하는 핵심 파라미터입니다. SHAP 분석에서 항상 가장 높은 영향도를 보입니다.

예측 결과 — 각 항목 설명

  • Kerf μm
    가공 후 Film 표면에 생기는 절단 선의 폭 예측값입니다. 실제 계측값과 비교하여 AI 예측 정확도를 확인할 수 있습니다.
  • Depth μm
    가공 후 Film에 남아있는 잔존 두께 예측값입니다. 이 값이 0μm 초과 25μm 이하이면 OK 판정입니다. 가장 중요한 예측값입니다.
  • 판정
    예측 Depth 기준으로 AI가 판단한 결과입니다. OK / 미가공 / 과가공 중 하나로 표시됩니다.
  • 신뢰도 %
    이 파라미터 조합에서 OK가 나올 통계적 확률입니다. 87%라면 같은 조건으로 100번 실험 시 약 87번 OK가 나온다는 의미입니다. 70% 이상이면 높은 신뢰도로 판단합니다.

SHAP 영향도 차트

AI가 이 파라미터를 선택한 이유를 설명하는 막대 차트입니다. 각 파라미터가 최종 판정에 얼마나 영향을 미쳤는지 비율로 보여줍니다.

📊 SHAP 차트 읽는 방법
  • 막대가 길수록 해당 파라미터가 이번 판정에 더 큰 영향을 미쳤습니다.
  • Power가 44%라면 이번 판정의 44%는 Power 값에 의해 결정됐다는 의미입니다.
  • 상위 5개 파라미터만 표시되며, 내림차순으로 정렬됩니다.
  • 경험과 SHAP 수치가 일치하면 AI 제안을 신뢰할 수 있는 신호입니다.
AI 설명 텍스트: 차트 아래에 LLM이 생성한 자연어 설명이 표시됩니다. "Power 15.2W에서 OK 확률이 87%입니다..." 형태로, 수치를 사람이 이해하기 쉬운 문장으로 변환한 것입니다.

레시피 재사용

한 번 OK가 확인된 파라미터는 레시피로 저장되어, 동일 소재 조합의 가공 시 즉시 활용할 수 있습니다.

🔄 레시피 매칭 우선순위
순위조건결과
1순위M1 길이 + M2 길이 + Thickness ±5μm 모두 일치정확 매칭 — 즉시 적용 권장
2순위M1 길이 + M2 길이만 일치 (Thickness 오차 > 5μm)유사 레시피 제공 — 참고용, 실험 권장
3순위해당 소재 조합 없음Auto DOE 신규 실행
유사 레시피 사용 시 주의: Thickness 오차가 5μm를 초과하는 유사 레시피는 참고용으로만 사용하세요. 실제 Depth가 OK 범위를 벗어날 수 있습니다. 중요한 가공이라면 Auto DOE를 다시 실행하는 것을 권장합니다.

자주 묻는 질문

AI가 제안한 Power 값이 경험과 다를 때 어떻게 하나요? ›
[수정 후 승인] 버튼을 이용하세요. AI 제안값을 경험에 기반한 값으로 바꾸고 승인할 수 있습니다. 수정된 내용과 실제 결과가 모두 기록되어 AI가 점차 개선됩니다.
신뢰도가 낮을 때(50% 이하)에도 실험해야 하나요? ›
반드시 실험할 필요는 없습니다. 신뢰도가 낮다는 것은 AI가 이 조건에 확신이 없다는 신호입니다. 거부하고 다시 제안을 요청하거나, 경험에 기반해 파라미터를 수정 후 승인하는 것을 권장합니다.
5~7회가 지나도 OK가 안 나오면 어떻게 하나요? ›
관리자에게 문의하세요. 소재 특성이 기존 학습 데이터와 크게 다른 경우일 수 있습니다. 관리자가 모델 재학습을 실행하거나 탐색 공간을 조정할 수 있습니다.
같은 소재인데 매번 Thickness를 새로 측정해야 하나요? ›
예, 매번 측정하는 것이 권장됩니다. Film 두께는 롤마다, 환경 조건에 따라 미세하게 달라집니다. 정확한 Thickness 입력이 AI 예측 정확도를 높입니다.
레시피가 이미 있어도 Auto DOE를 새로 실행할 수 있나요? ›
네. "저장된 레시피가 있습니다" 배너에서 [새로 탐색]을 선택하면 Auto DOE가 새로 실행됩니다. 기존 레시피가 의심스럽거나 소재 로트가 크게 변경된 경우 활용하세요.

증상원인해결방법
화면이 열리지 않음시스템이 기동되지 않았거나 네트워크 문제담당자에게 서버 상태 확인 요청
AI 제안이 2초 이상 걸림LLM 설명 생성 중 (정상)그대로 기다리세요. 최대 10초 이내 완료됩니다.
LLM 설명이 표시 안 됨Ollama 서비스 일시적 오류파라미터와 SHAP 차트는 정상 표시됩니다. 관리자에게 알려주세요.
결과 저장 시 오류 발생DB 연결 문제페이지를 새로고침 후 재시도. 반복되면 관리자에게 문의.
레시피가 안 나옴해당 소재 조합이 처음인 경우정상입니다. Auto DOE를 실행하여 새 레시피를 만드세요.
