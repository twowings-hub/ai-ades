# AI-ADES 시스템 사용 매뉴얼

이 문서는 AI 채팅이 "이 시스템 어떻게 써?" 류의 사용법 질문에 답할 때 참고하는 매뉴얼입니다.

## 1. 시스템 개요

AI-ADES(AI Autonomous Data Evaluation System)는 CO2 레이저로 Glass(M1) + Film(M2) 적층 소재를 절단할 때, AI가 최적의 레이저 가공 조건(Speed, Defocus, Frequency, Power)을 자동으로 찾아주는 시스템입니다.

- 기존에는 최적 조건을 찾기 위해 수동으로 30~50회 실험이 필요했지만, 이 시스템은 5~7회로 줄이는 것을 목표로 합니다.
- 운영자는 AI가 제안한 조건을 검토하고 승인 버튼만 누르면 됩니다(Human-in-the-Loop).
- AI는 베이지안 최적화(Bayesian Optimization) 기반 Auto DOE로 다음 실험 조건을 제안하고, XGBoost 모델로 Depth/Kerf와 품질 판정을 예측합니다.

## 2. 핵심 용어

- M1: 위쪽 소재(Glass 계열). M1 소재 종류와 길이(mm)를 입력합니다.
- M2: 아래쪽 소재(Film 계열). M2 소재 종류와 길이(mm)를 입력합니다.
- Thickness: 적층 소재의 실측 두께(μm).
- Speed: 레이저 이송 속도. Defocus: 초점 오프셋. Frequency: 펄스 주파수. Power: 레이저 출력(W). → AI가 찾는 4가지 가공 조건.
- Kerf: 절단폭(μm). Depth: 가공 깊이(μm).
- Auto DOE: AI가 다음에 시도할 실험 조건을 자동으로 설계·제안하는 기능.
- 레시피(Recipe): OK 판정을 받아 승인된, 소재 조합별 추천 가공 조건.

## 3. 판정 기준과 탐색 공간

판정 기준(Depth 기준):
- OK: Depth가 0μm 초과 ~ 25μm 이하 (AI의 목표)
- 과가공: Depth가 0μm (너무 깊게 가공되어 바닥까지 뚫림)
- 미가공: Depth가 25μm 초과 (덜 가공됨)
- NG: Defect(불량) 감지

탐색 공간(AI가 제안할 수 있는 값의 범위):
- Speed: 200, 500, 1000 중 하나
- Defocus: 0, 1, 2, 3, 4 중 하나
- Frequency: 100, 200 중 하나
- Power: 2.8 ~ 59.8(W) 사이 연속값
- AI는 이 범위를 벗어난 값을 제안하지 않습니다.

## 4. 전체 사용 흐름 (운영자 기준)

1. [실험 조건] 화면에서 M1/M2 소재 종류·길이와 Thickness를 입력하고 "Auto DOE 시작"을 누릅니다.
2. [AI 제안 승인] 화면에서 AI가 제안한 Speed/Defocus/Frequency/Power와 예측 결과(Depth, 판정, 신뢰도, SHAP 영향도)를 확인하고, 승인 / 수정 후 승인 / 거부 중 선택합니다.
3. 승인한 조건으로 실제 레이저 가공을 합니다.
4. [결과 보고] 화면에서 실측한 Kerf/Depth를 입력하고 저장합니다. "AI 평가" 버튼으로 결과 메모 초안을 자동 생성할 수 있습니다.
   - 저장 전 "실측 결과 확인" 모달에서 입력값과 사전 판정(예상 OK/미가공/과가공)을 확인합니다.
   - OK이면 레시피가 자동 저장되고 종료. OK가 아니면 "다음 제안 요청"으로 다음 조건을 받아 2번부터 반복합니다(보통 5~7회 내 수렴).
5. 이후 같은 소재 조합은 [레시피 조회]에서 저장된 레시피를 바로 확인할 수 있습니다.

## 5. 화면별 상세 사용법 (운영자 화면)

### 실험 조건
- M1 소재 종류와 M1 Length(mm), M2 소재 종류와 M2 Length(mm), Thickness(μm)를 입력합니다.
- M1 Length를 입력하면 M2 Length가 M1 × 2.5로 자동 채워집니다(필요 시 직접 수정).
- 프리셋 버튼(예: 4/10/20mm)으로 빠르게 값을 넣을 수 있습니다.
- 입력값이 학습 데이터 범위를 벗어나면 "외삽 경고"가 표시됩니다(예측 신뢰도 낮음, 탐색 횟수 증가 권장).
- 모든 값을 채우면 "Auto DOE 시작" 버튼이 활성화됩니다.

### AI 제안 승인
- AI 제안 파라미터(Speed/Defocus/Frequency/Power)와 예측 결과(예측 Kerf, 예측 Depth, 예측 판정, 신뢰도)를 보여줍니다.
- SHAP 영향도 그래프: 각 파라미터가 예측 Depth를 어느 방향으로 얼마나 바꾸는지 보여줍니다(오른쪽 파란색=Depth 증가, 왼쪽 빨간색=Depth 감소).
- 검증된 레시피가 있으면 "검증된 레시피 · 확인 실험" 배지가 표시됩니다.
- 버튼: "승인"(그대로 진행), "수정 후 승인"(파라미터를 직접 고치고 재예측 후 승인), "거부"(사유 입력 후 반려).

### 결과 보고
- 승인된 파라미터와 예측 결과를 표로 보여줍니다.
- 실측 Kerf(μm), Depth(μm)를 입력합니다. 입력하면 "AI 평가" 버튼으로 결과 설명 초안을 자동 생성할 수 있습니다(검토·수정 가능).
- "결과 저장" → "실측 결과 확인" 모달에서 사전 판정 미리보기를 확인하고 "확인 후 저장".
- OK면 레시피 저장 안내, OK가 아니면 "다음 제안 요청" 버튼으로 다음 Auto DOE 조건을 받습니다.
- 같은 세션의 Auto DOE 수렴 이력이 아래에 표로 누적됩니다.

### 레시피 조회
- M1/M2 소재 종류를 선택하고(길이·Thickness는 선택), "레시피 조회"를 누르면 저장된 레시피를 찾습니다.
- 길이를 비우고 소재 종류만 선택하면 그 조합의 모든 레시피를 보여줍니다.
- 전체 레시피 목록의 행을 클릭하면 입력 조건과 상세(소재 조건/레이저 가공 조건/예측·이력)에 반영됩니다.

### 실험 이력
- 지금까지 수행한 모든 실험의 입력값/가공 조건/측정 결과/설명을 조회합니다.
- 판정(OK/미가공/과가공/NG)으로 필터링하고, 실험 번호나 메모 내용으로 검색할 수 있습니다.

### AI 채팅
- 시스템에 저장된 실험 이력·레시피·소재별 현황 등 데이터에 대해 자연어로 질문하는 화면입니다.
- 우측 하단에서 답변에 쓸 LLM(Claude/OpenAI/Ollama)을 고를 수 있고, 좌측에서 과거 대화를 불러올 수 있습니다.
- 예: "M1 10mm + M2 25mm 조합의 OK 비율은?", "최근 실험 결과 요약해줘", "이 시스템 사용법 알려줘".

## 6. 관리자 콘솔 사용법

관리자(admin) 권한으로 접근하는 시스템 설정 화면입니다. 메뉴별 기능:

- 사용 안내: 관리자 메뉴 용도와 주의사항, 운영 흐름 요약(인수인계용).
- 시스템 상태: 서비스 헬스체크(ok/degraded/down), 최근 모델 학습 지표(R²/F1), CPU·RAM·Disk 사용량. 30초마다 자동 갱신.
- 서비스 관리: 각 서비스(컨테이너) 재시작. execution-agent(자기 자신)는 재시작 불가.
- LLM 모델 선택: AI 설명/채팅에 쓰는 LLM 프로바이더·모델 전환 + 연결 테스트. 폐쇄망에서는 ollama(로컬)만 사용.
- 모델 재학습: 누적 실험 데이터로 모델 재학습(수동 시작 + 누적 임계값 자동). 학습은 수 분 소요.
- 판정 기준 설정: OK 판정 범위(Depth)와 Auto DOE 탐색 공간 조정. 공정 핵심값이므로 합의 없이 변경 금지.
- 소재 종류 관리: 실험 조건 화면의 M1/M2 소재 선택 목록 추가·수정·활성화. 비활성화하면 목록에서 빠짐.
- 사용자 관리: 운영자/관리자 계정 추가·역할 변경·비밀번호 재설정.
- 알림 설정: 수신 메일 주소와 알림 이벤트(OK 판정/가공 실패/모델 성능 저하) 선택.
- 메일 서버 설정: 알림 메일을 보낼 SMTP 서버 설정. 폐쇄망에서는 사내 메일서버만 사용. 저장값이 즉시 반영되며, 비우면 메일은 발송되지 않고 알림은 Grafana·기록으로만 남음.
- 감사 로그: 승인/거부/설정 변경/재학습/데이터 삭제 등 모든 관리 작업 이력 조회·CSV 내보내기.
- 데이터 관리: DB 백업, 실험 CSV 내보내기, 학습 데이터 Excel 업로드, 테스트 데이터 정리. 테스트 데이터 삭제는 영구이며 되돌릴 수 없음.

## 7. 자주 묻는 질문(FAQ)

- 실험을 몇 번 하면 되나요? 보통 5~7회 내에 OK 조건으로 수렴합니다. 외삽(학습 범위 밖) 입력이면 더 걸릴 수 있습니다.
- AI 제안을 그대로 따라야 하나요? 아니요. "수정 후 승인"으로 직접 값을 바꿔 재예측할 수 있습니다. 단, 탐색 공간 범위는 지키세요.
- OK가 안 나오면? "다음 제안 요청"을 누르면 AI가 이전 결과를 반영해 다음 조건을 제안합니다.
- 같은 소재를 또 가공하려면? [레시피 조회]에서 저장된 레시피를 찾아 그대로 사용하면 됩니다(확인 실험 1회 권장).
- 알림 메일이 안 와요. 관리자 → 메일 서버 설정의 SMTP가 되어 있는지, 알림 설정에 수신 메일과 이벤트 체크가 되어 있는지 확인하세요. 폐쇄망에서는 외부(Gmail 등) 메일이 막힐 수 있습니다.

## 8. 주의사항

- 판정 기준값(0μm/25μm)과 Auto DOE 탐색 공간은 공정 핵심값입니다. 합의 없이 바꾸면 AI 제안·판정이 전부 달라집니다.
- 테스트 데이터 정리의 삭제는 영구이며 되돌릴 수 없습니다.
- 폐쇄망(외부 접속 불가) 환경에서는 외부 메일/외부 LLM이 동작하지 않습니다. 사내 SMTP와 로컬 ollama를 사용하세요.
